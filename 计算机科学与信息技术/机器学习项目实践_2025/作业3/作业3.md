# 程序1



为了确定kNN算法在鸢尾花数据集上分类的最优k值，可以采用交叉验证和网格搜索的方法。以下是具体步骤和代码实现：

步骤说明

1. 数据准备：加载鸢尾花数据集，包含150个样本和4个特征。
2. 标准化：由于kNN对特征尺度敏感，需对数据进行标准化处理。
3. 参数搜索：定义k的候选范围（如1到30的奇数），避免平票问题。
4. 交叉验证：通过网格搜索（GridSearchCV）结合交叉验证（5折）评估不同k值的分类准确率。
5. 最优结果：选择平均准确率最高的k值作为最优解。

代码实现

python

复制

```python
from sklearn.datasets import load_iris
from sklearn.model_selection import GridSearchCV
from sklearn.neighbors import KNeighborsClassifier
from sklearn.pipeline import Pipeline
from sklearn.preprocessing import StandardScaler

# 加载鸢尾花数据集
iris = load_iris()
X, y = iris.data, iris.target

# 创建Pipeline：先标准化，再kNN分类
pipeline = Pipeline([
    ('scaler', StandardScaler()),    # 标准化处理
    ('knn', KNeighborsClassifier())  # kNN模型
])

# 定义k值的候选范围（1到30的奇数）
param_grid = {
    'knn__n_neighbors': list(range(1, 31, 2))
}

# 网格搜索（5折交叉验证）
grid_search = GridSearchCV(
    estimator=pipeline,
    param_grid=param_grid,
    cv=5,
    scoring='accuracy'
)

# 执行搜索
grid_search.fit(X, y)

# 输出最优结果
best_k = grid_search.best_params_['knn__n_neighbors']
best_score = grid_search.best_score_
print(f"最优k值: {best_k}")
print(f"交叉验证平均准确率: {best_score:.4f}")

# 可选：查看不同k值的表现
results = grid_search.cv_results_
for i in range(len(results['params'])):
    k = results['params'][i]['knn__n_neighbors']
    score = results['mean_test_score'][i]
    print(f"k={k:<2} | 平均准确率: {score:.4f}")
```

结果分析
 • 最优k值：通常会在较小的奇数中出现（如3、5、7），因为鸢尾花数据集类别间区分度较高。

• 准确率变化：k值过小可能导致过拟合，k值过大会忽略局部特征。交叉验证结果会显示准确率随k值变化的趋势。

示例输出

```
最优k值: 3
交叉验证平均准确率: 0.9733
k=1  | 平均准确率: 0.9533
k=3  | 平均准确率: 0.9733
k=5  | 平均准确率: 0.9667
...
k=29 | 平均准确率: 0.9267
```

关键点总结
 • 标准化必要性：kNN依赖距离计算，标准化可消除特征量纲影响。

• 交叉验证：避免单次划分数据的偶然性，提升结果可靠性。

• k值范围：从1开始逐步增加，观察模型是否欠拟合或过拟合。

通过上述方法，可以快速找到在鸢尾花数据集上表现最佳的k值。

import pandas as pd
from sklearn.model_selection import train_test_split

# 加载数据集（假设本地文件为diabetes.csv）
url = "https://raw.githubusercontent.com/jbrownlee/Datasets/master/pima-indians-diabetes.data.csv"
df = pd.read_csv(url, header=None)
df.columns = ["Pregnancies", "Glucose", "BP", "SkinThickness", "Insulin", "BMI", "DPF", "Age", "Outcome"]

**5. 关键优化方向**

1. 数据质量
    • 处理缺失值和异常值（如糖尿病数据中的0值）。

   • 标准化或归一化特征（必选项）。

2. 模型调参
    • 使用网格搜索和交叉验证选择最佳k值。

   • 尝试不同距离度量（`metric`参数）和加权投票（`weights`参数）。

3. 特征工程
    • 通过统计检验（如ANOVA）或模型（如随机森林）筛选重要特征。

   • 降维技术（如PCA）减少冗余信息。

4. 类别不平衡处理
    • 过采样（SMOTE）、欠采样或调整类别权重。

5. 集成方法
    • 结合多个kNN模型（如Bagging）提升鲁棒性。

**6. 完整代码示例**

python

复制

```python
import pandas as pd
from sklearn.model_selection import train_test_split, GridSearchCV
from sklearn.preprocessing import StandardScaler
from sklearn.neighbors import KNeighborsClassifier
from sklearn.metrics import accuracy_score, classification_report
from imblearn.over_sampling import SMOTE
from sklearn.feature_selection import SelectKBest, f_classif

# 加载数据
url = "https://raw.githubusercontent.com/jbrownlee/Datasets/master/pima-indians-diabetes.data.csv"
df = pd.read_csv(url, header=None)
df.columns = ["Pregnancies", "Glucose", "BP", "SkinThickness", "Insulin", "BMI", "DPF", "Age", "Outcome"]

# 处理缺失值
for col in ["Glucose", "BP", "SkinThickness", "Insulin", "BMI"]:
    df[col] = df[col].replace(0, df[col].mean())

# 划分数据集
X = df.drop("Outcome", axis=1)
y = df["Outcome"]
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# 标准化
scaler = StandardScaler()
X_train_scaled = scaler.fit_transform(X_train)
X_test_scaled = scaler.transform(X_test)

# 优化1：选择最优k值
param_grid = {"n_neighbors": range(1, 20, 2)}
grid = GridSearchCV(KNeighborsClassifier(), param_grid, cv=5, scoring="accuracy")
grid.fit(X_train_scaled, y_train)
best_k = grid.best_params_["n_neighbors"]

# 优化2：特征选择
selector = SelectKBest(score_func=f_classif, k=5)
X_train_selected = selector.fit_transform(X_train_scaled, y_train)
X_test_selected = selector.transform(X_test_scaled)

# 优化3：类别平衡（SMOTE）
smote = SMOTE(random_state=42)
X_resampled, y_resampled = smote.fit_resample(X_train_selected, y_train)

# 最终模型
knn = KNeighborsClassifier(n_neighbors=best_k, weights="distance")
knn.fit(X_resampled, y_resampled)
y_pred = knn.predict(X_test_selected)

# 评估
print("优化后准确率:", accuracy_score(y_test, y_pred))
print("分类报告:\n", classification_report(y_test, y_pred))
```
