## 被测函数 `DateUtils.java`

java

复制

```java
public class DateUtils {
    /**
     * 计算输入日期的下一天
     * @param year 年（1900-2050）
     * @param month 月（1-12）
     * @param day 日（1-31）
     * @return 下一天的日期字符串，格式为 "yyyy-MM-dd"
     */
    public static String getNextDate(int year, int month, int day) {
        int[] monthDays = {31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};
        boolean isLeapYear = (year % 4 == 0 && year % 100 != 0) || (year % 400 == 0);
        
        // 处理闰年2月
        if (isLeapYear && month == 2) {
            monthDays[1] = 29;
        }

        // 检查是否为最后一天
        if (day >= monthDays[month - 1]) {
            day = 1;
            if (month == 12) {
                month = 1;
                year++;
            } else {
                month++;
            }
        } else {
            day++;
        }

        return String.format("%04d-%02d-%02d", year, month, day);
    }
}
```

## 测试类 `DateUtilsTest.java`

java

复制

```java
import org.junit.jupiter.api.Test;
import org.junit.jupiter.params.ParameterizedTest;
import org.junit.jupiter.params.provider.CsvSource;
import static org.junit.jupiter.api.Assertions.assertEquals;

public class DateUtilsTest {

    // 边界值分析法测试用例
    @ParameterizedTest
    @CsvSource({
        // 输入：year, month, day | 预期输出
        "2000, 1, 31, 2000-02-01",   // 1月最后一天 → 跨月
        "2000, 12, 31, 2001-01-01",  // 12月最后一天 → 跨年
        "2000, 2, 29, 2000-03-01",   // 闰年2月结束
        "1900, 2, 28, 1900-03-01",   // 非闰年2月结束
        "1900, 12, 31, 1901-01-01",  // 最小年份跨年
        "2050, 12, 31, 2051-01-01"   // 最大年份跨年
    })
    void testBoundaryValues(int year, int month, int day, String expected) {
        String actual = DateUtils.getNextDate(year, month, day);
        assertEquals(expected, actual);
    }

    // 判定表驱动法测试用例
    @ParameterizedTest
    @CsvSource({
        // 输入：year, month, day | 预期输出 | 覆盖规则
        "2000, 5, 10, 2000-05-11",  // 规则1：非最后一天
        "2000, 4, 30, 2000-05-01",  // 规则2：非12月最后一天 → 跨月
        "2000, 12, 31, 2001-01-01", // 规则3：12月最后一天 → 跨年
        "2000, 2, 29, 2000-03-01",  // 规则4：闰年2月结束
        "1900, 2, 28, 1900-03-01"   // 规则5：非闰年2月结束
    })
    void testDecisionTable(int year, int month, int day, String expected) {
        String actual = DateUtils.getNextDate(year, month, day);
        assertEquals(expected, actual);
    }
}
```

------

## 代码说明

### 1. **被测函数逻辑**

- **闰年判断**：能被4整除且非整百年，或能被400整除的年份为闰年。

- **月份天数处理**：使用数组 `monthDays` 存储各月天数，动态调整闰年2月的天数。

- 

  日期计算

  ：

  - 如果是最后一天，重置日为1，并根据月份判断是否跨年。
  - 否则直接递增日。

### 2. **测试代码设计**

- **边界值测试**：覆盖月份、日、年的极值（如1月、12月、闰年2月等）。
- **判定表测试**：覆盖规则组合（最后一天、跨月、跨年等逻辑分支）。
- **参数化测试**：通过 `@CsvSource` 提供多组输入和预期输出。
