# 题目案例

### **题目 2：字典频率统计**

**描述**：
给定一个字符串，统计每个字符出现的次数，并按频率降序输出前 k 个字符及其次数。若频率相同，则按字典序排列。
**输入**：`s = "abracadabra", k = 3`
**输出**：`[('a', 5), ('b', 2), ('r', 2)]`



**难度**：★★★☆☆
**参考答案**：

```go
import (
    "container/heap"
    "sort"
)

func topKFrequent(s string, k int) []pair {
    freq := make(map[rune]int)
    for _, c := range s {
        freq[c]++
    }

    pairs := make([]pair, 0, len(freq))
    for c, cnt := range freq {
        pairs = append(pairs, pair{c, cnt})
    }

    sort.Slice(pairs, func(i, j int) bool {
        if pairs[i].count == pairs[j].count {
            return pairs[i].char < pairs[j].char
        }
        return pairs[i].count > pairs[j].count
    })

    return pairs[:k]
}

type pair struct {
    char  rune
    count int
}
```





### 7-8 字符串去重排序

分数 10

全屏浏览切换布局

作者 刘琦

单位 天津城建大学

输入一个非空字符串，去除重复的字符后，从小到大排序，输出为一个新字符串。

```go
package main

import (
	"bufio"  // 用于高效读取输入
	"fmt"    // 格式化输入输出
	"os"     // 操作系统交互，如读取标准输入
	"sort"   // 提供排序功能
	"strings" // 字符串处理函数
)

func main() {
	// 读取输入字符串并去除首尾空白
	reader := bufio.NewReader(os.Stdin)
	input, _ := reader.ReadString('\n')
	input = strings.TrimSpace(input)

	// 使用map去重：map的键具有唯一性，值设为true即可
	charMap := make(map[rune]bool)
	for _, char := range input {
		charMap[char] = true // 重复的字符会自动覆盖
	}

	// 将map的键（不重复的字符）转换为rune切片
	chars := make([]rune, 0, len(charMap))
	for char := range charMap {
		chars = append(chars, char)
	}

	// 对字符切片进行升序排序（按Unicode码点）
	sort.Slice(chars, func(i, j int) bool {
		return chars[i] < chars[j]
	})

	// 将排序后的rune切片转换为字符串并输出
	result := string(chars)
	fmt.Println(result)
}
```

Solidity 提供了四种不同的函数可见性修饰符：

- **public**：可以从外部和合约内部访问。
- **private**：只能从合约内部访问。
- **internal**：只能从当前合约及其派生合约访问。
- **external**：只能从外部访问，不能从合约内部调用。

```solidity
contract Example {
    function publicFunc() public view returns (uint) { return 1; }
    function privateFunc() private pure returns (uint) { return 2; }
    function internalFunc() internal pure returns (uint) { return 3; }
    function externalFunc() external pure returns (uint) { return 4; }
}
```

- **payable**：允许函数接收以太币。

## 简易程序案例



```solidity
// SPDX-License-Identifier: MIT  // 指定合约的开源许可证为 MIT
pragma solidity ^0.8.0;  // 指定编译此合约所需的 Solidity 版本

contract SimpleToken {
    // 代币的基本信息
    string public name = "SimpleToken";  // 代币名称
    string public symbol = "ST";  // 代币符号
    uint8 public decimals = 18;  // 代币精度（18 位小数，与 ETH 相同）
    uint256 public totalSupply;  // 代币总供应量
    mapping(address => uint256) public balanceOf;  // 地址到余额的映射

    // 合约构造函数，部署时执行
    constructor(uint256 _initialSupply) {
        // 计算初始总供应量（考虑小数位）
        totalSupply = _initialSupply * (10 ** uint256(decimals));
        // 将初始代币全部分配给合约部署者
        balanceOf[msg.sender] = totalSupply;
    }

    // 代币转账函数
    function transfer(address _to, uint256 _value) public returns (bool success) {
        // 检查发送者余额是否足够
        require(balanceOf[msg.sender] >= _value, "余额不足");
        // 从发送者账户扣除代币
        balanceOf[msg.sender] -= _value;
        // 向接收者账户添加代币
        balanceOf[_to] += _value;
        return true;
    }

    // 铸造新代币的函数（增发）
    function mint(address _to, uint256 _value) public {
        // 权限检查：仅允许管理员地址（0x1）铸造代币
        require(msg.sender == address(0x1), "只有管理员可以铸造代币");
        // 增加总供应量
        totalSupply += _value;
        // 向指定地址添加代币
        balanceOf[_to] += _value;
    }
}
```





### 4.1 基本类型

#### 整数类型：

- `uint` 和 `int`：分别表示无符号整数和有符号整数。
  - `uint8`, `uint16`, ..., `uint256`：从 8 到 256 位的无符号整数。
  - `int8`, `int16`, ..., `int256`：从 8 到 256 位的有符号整数。

```solidity
uint public myUint = 10;
int public myInt = -10;
```

#### 地址类型：

- `address`：用于存储以太坊账户地址。
  - `address payable`：可接收以太币的地址。

```solidity
address public owner;
address payable public receiver;
```

#### 布尔类型：

- `bool`：用于存储布尔值（true 或 false）。

```solidity
bool public isActive = true;
```

#### 字符串类型：

- `string`：用于存储任意长度的字符串。

```solidity
string public name = "Alice";
```
